# 소개

[![install size](https://packagephobia.com/badge?p=@gdsc-dju/styled-components-theme)](https://packagephobia.com/result?p=@gdsc-dju/styled-components-theme-theme)

GDSC DJU 웹서비스의 `colors`, `font-sizes`, `window-sizes` 에셋을 쉽게 설정하기 위한 라이브러리입니다.

## 설치하기

:::caution

이 라이브러리는 `styled-components`에 의존성을 지니고 있습니다. 꼭 설치해주세요.

:::




```shell
yarn add styled-components @gdsc-dju/styled-components-theme
```
